# prakweb3
Head: untuk ngatur judul web, manggil font dari Google, icon Font Awesome, dan menghubungkan file CSS.

Body: tampilan halaman dan diatur supaya konten ada di tengah layar.

Container: pembungkus atau kotak utama untuk semua elemen biar layout rapi dan tersusun ke bawah.

Profil: isinya adalah foto profil dan nama, fotonya dibikin bulat biar keliatan lebih estetik.

Social: Bagian ini adalah bagian yang menampilkan judul dan kumpulan link media sosial.

Sosmed: sebagai tombol link ke Instagram, LinkedIn, dan GitHub, dengan efek berubah warna saat diklik.

CSS: CSS dipakai buat mengatur ukuran, warna, font, posisi, dan background supaya tampilan web lebih menarik.
